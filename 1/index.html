Who runs the internet?

Nobody

(The lunatics are running the asylum)

By General Agreement, the players:

  1. Carriers have peering arrangements to deliver bandwidth (Comcast, Verizon)

  2. Standards groups promote “best” practices (W3C, WHATWG)

  3. Browser Makers implement whatever they want (Microsoft, Mozilla, Google, Opera)

  4. Servers serve pages (Apache, IIS, Nginx)

  5. Scripting languages talk to servers (PHP, C#, Python)

How do you get your stuff online?

  • This is called deployment

  • Normally you'll FTP your site to a host

  • We're going to use GitHub - way cooler

    • let's you version control your code

    • you can host static pages

    • you will merge in new homework

Let's try it now - create a GitHub account and launch a page

  • go to www.GitHub.com

  • choose your username and pick a password

Forking

  • go to https://github.com/cykod/web304-spring2012

  • click the fork button

  • Click Admin and change the repository name to [yourusername].github.com

  • (Click back to Admin) Next click on "Collaborators," type in "cykod" and click add

  • open or download GitHub for Mac
    (store this on a key to make your life easier)

GitHub for Mac

  • Open it

  • Put in your login info

  • Find your "username.github.com" repository

    • click on "clone"

    • clone it to your key or folder you can access later

    • Double click on it

    • click "open in finder"

    • open the "1" folder and edit the index.html file in textmate

    • go back to GitforMac and commit your changes

    • sync your repository

Your Done!
